:w
Write file, don't exit
:wa
Write all files, don't exit
:w filename
Write to file filename, don't exit
:wq
Write file, exit
:xa
Write all files, exit
:x,y w filename
Write lines x through y to file filename
:x,y w >> filename
Append lines x through y to file filename
:r filename
Insert contents of file filename
:xr filename
Insert contents of file filename under line x
:e .
Open built-in file explorer
:Sex
Split window and open built-in file explorer
:browse e
Open graphical file explorer
:cd directory
Change directory to directory
:q
Quit (if nothing changed)
:qa
Quit all (if nothing changed)
:q!
Quit (abort changes)
:qa!
Quit all (abort changes)
:sh
Drop to shell (execute commands)
:! command
Execute command command
:%! command
Execute command command, insert output
:%!xxd
Enter hex mode
:%!xxd -r
Exit hex mode
ggVGg?
Change text to ROT13
:syntax on
Turn on syntax highlighting
:syntax off
Turn off syntax highlighting

Set Commands
Enter :set followed by a parameter:

number or nonumber
Turn on/off line numbering
wrap or nowrap
Turn on/off word wrap
autoindent or noautoindent
Turn on/off autoindent
smartindent or nosmartindent
Turn on/off smart autoindent
shiftwidth=x
Set indentation to x spaces
ff=unix
Convert to UNIX file format
ff=dos
Convert to DOS file format
foldmethod=manual
Folds must be manually defined (see Code Folding section)
foldmethod=indent
Folds are automatically created based on indentation level
foldmethod=expr
Folds are defined by a user-defined expression
foldmethod=syntax
Folds are defined by syntax highlighting
nofoldenable
Turn off folding
foldnestmax=x
Set the deepest fold to x levels
foldlevel=x
Set the fold level to x (0 = all folds open)

Code Folding

zf
Create fold under cursor
zd
Delete fold under cursor
zD
Delete all folds
zE
Delete all folds in file
zo
Open fold under cursor
zc
Close fold under cursor
zO
Open all folds
zC
Close all folds
za
Toggle open/close fold
xzF
Fold x lines
:fold
Fold selected lines (visual mode)
zj
Move cursor to next fold
zk
Move cursor to previous fold
zm
Increase fold level by one
zr
Decrease fold level by one
zR
Decrease fold level to zero (all folds open)
